Modelado de Funciones Lineales

15/03/2023 Matemáticas Avanzadas

Objetivo

El objetivo principal de esta actividad fue aplicar los conceptos de funciones lineales para modelar un conjunto de datos reales y poder realizar predicciones basadas en el modelo obtenido.

Desarrollo

Se recopilaron datos sobre el crecimiento de plantas bajo diferentes condiciones de luz. Utilizando el método de mínimos cuadrados, determinamos la ecuación de la recta que mejor se ajustaba a los datos obtenidos.

El proceso incluyó:

  • Recolección y organización de datos
  • Cálculo de pendiente e intercepto
  • Validación del modelo
  • Interpretación de resultados

Resultados

Se obtuvo un modelo con un coeficiente de determinación (R²) de 0.92, lo que indica un buen ajuste a los datos. El modelo permitió predecir con precisión el crecimiento de las plantas en diferentes condiciones de iluminación.
